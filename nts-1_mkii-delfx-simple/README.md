# Simple
Tempo sync delay effect,  
can use as `DELAY EFFECTS`.

# Parameters
- time
  - 0.125 (1/8)
  - 0.1666
  - 0.1875 (3/16)
  - 0.25 (1/4)
  - 0.3333
  - 0.375 (3/8)
  - 0.5 (1/2)
  - 0.6666
  - 0.75 (3/4)
  - 1.0
- depth  
feedback gain.
- shift depth  
dry/wet mixing ratio.

# How to build
1. Clone (or download) this project.
1. Copy to under `platform/nts-1_mkii`.
1. See README.md under `platform/nts-1_mkii`.
